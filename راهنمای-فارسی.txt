فایل راهنمای کتابخانه فرازش ریسپانسیو
این کتابخانه صرفا برای ریسپانسیو صفحات وب می باشد
شما باید از کلاس های آماده css در کد های html خود استفاده کنید
شما با این کتابخانه میتوانید حداکثر 20 ستون در یک ردیف به نمایش بزارید

هر ظرف را با dish و هر سطون‌ را داخل Row قرار میدیم مثال :
<div class="dish">
   <div class="row">
کد های این ستون شما        
   </div>
</div>


برای یادگیری کامل لطفا نقاط شکست را مشاهده کنید

نقاط شکست (breaking points)

دستگاه های بسیار کوچک دستگاه هایی هستند که صفحاتشان 576 پیکسل کوچکتر است
این دسته حروف اختصاری ندارد ( پیش فرض )


دستگاه های کوچک دستگاه هایی هستند که صفحاتشان 576 پیکسل یا بزرگ تر است
این دسته با حروف اختصاری زیر
sm
در کتابخانه به کار میرود



دستگاه های متوسط دستگاه هایی هستند که صفحاتشان 768 پیکسل یا بزرگ تر است
این دسته با حروف اختصاری زیر
med
در کتابخانه به کار میرود

دستگاه های بزرگ دستگاه هایی هستند که صفحاتشان 992 پیکسل یا بزرگ تر است
این دسته با حروف اختصاری زیر
lg
در کتابخانه به کار میرود

دستگاه های بسیار بزرگ دستگاه هایی هستند که صفحاتشان 1200 پیکسل یا بزرگ تر است
این دسته با حروف اختصاری زیر
xl
در کتابخانه به کار میرود

اگر نقاط شکست را مشاهده کرده اید و حروف اختصاری آنها را به خوبی به خاطر سپرده باشید به مثال های زیر نگاه کنید
box-sm-? , box-med-? , box-lg-? , box-xl-? , box-vl-?
اگر دقت کرده باشید میتوان هر اندازه ای برای هر دستگاهی در نظر گرفت

هر یک از حروف اختصاری کلا وسط box-? قرار میگیرد
box-5 برای دستگاه medium => box-med-5
box-5 برای دستگاه small => box-sm-5


استفاده از کلاس های عرض و طول
h => height || w => width
تا 100 درصد امکان استفاده وجود دارد

مثال :
class ( w-5 ) == css ( width: 5% )
class ( h-5 ) == css ( height: 5%) l
به جای 5 درصد مورد نظر شما قرار میگیرد
به شرح ساده 
w-? / h-?
به جای علامت ? درصد مورد نظر خودتون رو بزارید از 1 تا 100
امکان استفاده آنها در سایز های مختلف با استفاده از نقطه های شکست 
مثال
h-med-5 / w-lg-6
